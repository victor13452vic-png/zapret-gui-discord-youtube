## 📖 Описание

**Zapret Gui** — это удобный графический интерфейс для обхода DPI блокировок. Программа делает обход блокировок Discord, YouTube и других сервисов максимально простым: запустил, выбрал стратегию, нажал кнопку.

В основе лежит мощный движок [Zapret](https://github.com/Flowseal/zapret-discord-youtube) от Flowseal, обёрнутый в современный и понятный интерфейс.

## ✨ Возможности

- 🎨 **Красивый современный интерфейс** в зелёных тонах
- 🚀 **12 встроенных стратегий обхода** (general + ALT 1-11)
- 🧪 **Тестирование стратегий** — автоматически проверяет какая работает лучше всего и применяет её
- ⚙️ **Настройки в одном окне**:
  - Автозапуск с Windows
  - Авто-подключение после перезагрузки
  - Игровой фильтр (TCP/UDP режимы)
  - IPSet фильтр (loaded/none/any)
  - Обновление списков hosts и ipset одной кнопкой
- 📦 **Простой установщик** — никаких ручных настроек

## 📥 Установка

Скачайте `ZapretGui-Setup.exe` из [Releases](../../releases) и запустите.

Установщик сделает всё сам:
- Создаст папку программы
- Распакует все файлы
- Создаст ярлыки на рабочем столе и в меню Пуск

## 🎯 Использование

1. **Выберите стратегию** в правой панели (например `general` или `general (ALT)`)
2. Нажмите **большую круглую кнопку** в центре для подключения
3. При первом запуске рекомендуется зайти в **⚙ Настройки** → **Тест стратегий** — программа сама подберёт оптимальную
